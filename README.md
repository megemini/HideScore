# HideScore

屏蔽掉赛事（足球、篮球）录像页面（咪咕、腾讯）比分的脚本插件 ～～～ 

Chrome、Via 可用 🤗🤗🤗 

使用前：

![image](doc/before.jpg)

使用后：

![image](doc/after.jpg)

## 使用方法

### Chrome

1. 打开 chrome://extensions/
2. 打开 `Developer mode`
3. 加载插件 `load unpacked`，选择整个文件夹

![image](doc/chrome.jpg)

### Via

1. 打开设置

![image](doc/via_0.jpg)

2. 选择 `脚本`

![image](doc/via_1.jpg)

3. 选择 `导入脚本` ，浏览手机文件，选择 `content_via.js`

![image](doc/via_2.jpg)

4. 确认安装

![image](doc/via_3.jpg)

## 注意

目前只测试了咪咕、腾讯的录像比分页面。

Via 使用的时候需要使用 `电脑模式`

🤗🤗🤗
